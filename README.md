# SewaCar

SewaCar is a car rental web application built with **Next.js**. It allows users to browse available cars, view car details, and manage bookings. Admins can manage users, cars, and availability through a dedicated dashboard.

## Features

- Browse cars with search, filter, and sorting options
- View detailed car information, including availability and owner
- Admin dashboard to manage users, cars, and car availability
- Interactive charts for car statistics and user statistics
- Responsive design for desktop and mobile
